# CPU Scheduling Simulator
A Python GUI application for simulating and comparing CPU scheduling algorithms.

## Features
* First Come First Serve (FCFS)
* Shortest Job First (SJF)
* Round Robin (RR)
* Priority Scheduling
* Gantt Chart Visualization
* Waiting Time, Turnaround Time, and CPU Utilization Metrics
* Algorithm Comparison

## Requirements
```bash
pip install customtkinter matplotlib
```

## Running the app
```bash
python app.py
```
